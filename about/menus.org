* Menu Generation

Menus are generated by including this =header.org= file containing an
[[https://www.gnu.org/software/emacs/manual/eintr.html][Emacs Lisp]]
=menu= code block. =menu= code inserts the GitHub URL and adjusts the
relative =href= for each menu
item. [[https://www.gnu.org/software/emacs/manual/html_mono/eintr.html][(Emacs Lisp Doc)]]
